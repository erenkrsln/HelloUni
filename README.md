1) ``npm install``

hier choose existing project, HelloUni Team und webapp wählen, cloud deployment

ggfs noch ``npx auth secret`` (setzt benötigtes secret)
4) ``npm run dev`` <br>
-> sollte auf port 3000 laufen, convex dashboard development (nur lokale posts)
