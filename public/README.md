# Public Assets

Lade hier dein Logo hoch:

1. Benenne dein Logo-Bild `logo.png` (oder `logo.jpg`, `logo.svg`)
2. Lege es in diesen `public/` Ordner
3. Das Logo wird automatisch oben rechts im Header angezeigt

Unterstützte Formate: PNG, JPG, SVG

Das Logo sollte idealerweise 61x68px groß sein (wie im Design spezifiziert).
